 Sales Data Analysis Project using Excel, SQL, and Power BI

## Project Overview

This project focuses on analyzing retail sales data to identify business trends, customer behavior, product performance, and regional sales distribution. The project demonstrates end-to-end data analysis skills, including data cleaning, exploratory data analysis, SQL querying, and dashboard development.

The analysis was performed using Microsoft Excel, SQL, and Power BI to generate meaningful business insights and create an interactive dashboard.

---

## Objectives

The main objectives of this project were:

* Analyze sales and profit performance.
* Identify high-performing regions and product categories.
* Analyze customer purchasing behavior.
* Study sales trends over time.
* Create an interactive dashboard for business decision-making.
* Demonstrate practical data analysis skills using industry-standard tools.

---

## Dataset Information

* Dataset Name: Sample Superstore Dataset
* Dataset Type: Retail Sales Dataset
* Total Records: 9,994 rows
* Total Columns: 21 columns

The dataset contains information related to:

## Key Business Insights

Some important insights obtained from the analysis:

* West regions generated significantly higher sales than others.
* Technology category performance varied across regions.
* Sales showed trends over multiple years.
* A small number of customers contributed a large share of revenue.
* Office supplies products generated lower profits despite strong sales performance.
